# State Medical Licensing Research — March 2026

## Key Data Points for the Licensing Guide

### Processing Speed Categories
- Fast (2-6 weeks): CO, FL, IN, MT, NE, UT
- Medium (6-12 weeks): Most IMLC states, OH, MI, GA, NY
- Slow (4-8+ months): CA, MA, TX, NV, IL

### GME Requirements by State
- 1 year minimum: some states
- 2 years: 12 states
- 3 years: 25 states (including CA for ALL applicants)

### IMLC Member States: 44 jurisdictions (42 states + DC + Guam)
- NOT members: California, Florida
- NY: passed legislation but not yet implemented
- IMLC fee: $700 + individual state license fees

### License Fees
- Lowest: PA $35
- Highest: NV $1,425
- Average: ~$500
- CA total: ~$1,299

### Jurisprudence Exam Required: TX, ME, AL

### FCVS
- Initial: $395 (includes 1 profile)
- Additional profiles: $99 each
- Processing: ~35 days initial, ~20 days subsequent
- Worth it: YES, especially for multi-state

### DEA Registration
- Fee: $888 for 3 years
- Process: 4-6 weeks
- Need state license first
- Separate registration per state

### CDS Required States (~23): AL, CT, DE, DC, HI, ID, IL, IN, IA, LA, MD, MA, MI, MO, NV, NJ, NM, OK, RI, SC, SD, UT, WY

### IMG Additional Requirements
- ECFMG certification (mandatory everywhere)
- ECFMG Certification Status Report ($66/state)
- Some states: 3 years GME (vs 1 for US grads)
- Document translation if not English
- Visa documentation
- 18+ states now have alternative pathways without US residency

### Common Mistakes
1. Incomplete applications
2. Unexplained timeline gaps (30+ days)
3. Failure to disclose (even minor issues)
4. Not chasing third-party verifications
5. Waiting until last minute
