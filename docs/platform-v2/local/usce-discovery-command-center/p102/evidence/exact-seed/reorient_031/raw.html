Visiting Medical Students

Visiting Medical Students

Visiting Resident Request to Rotate at Atrium Health - Carolinas Medical Center

  • Rotation requests must be submitted 90 days or more in advance of the anticipated start date
  • There is a $250 application fee which must be paid one month prior to the start date
  • The 90-day process starts when the GME Office receives Visiting Resident Policy - Attachment I from the Program Coordinator  
  • Send an email request to the Program Coordinator of the specialty you would like to rotate with
