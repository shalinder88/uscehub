How we verify

A directory is only as good as its links.

USCEHub runs an automated cron daily against every approved listing's source URL and routes failures to a human admin queue within 24 hours. A listing whose source link returns an error is flagged "Needs recheck" — never silently hidden. If the institution has retired the program, we mark "Program closed" and stop showing it in default search results.

Reviews are user-submitted feedback, moderated before publishing, and shown separately from source-link verification. Reviews are opinions; verification is an audit. We do not let one substitute for the other.

USCEHub is an independent informational platform. We are not affiliated with NRMP, ECFMG, ERAS, AAMC, or any hospital or residency program. We do not guarantee placement, match, or visa outcomes. Always verify program details with the institution before applying.
