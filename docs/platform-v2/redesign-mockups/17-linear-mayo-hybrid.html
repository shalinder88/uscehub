How we verify

A directory is only as good as its links.

USCEHub runs an automated cron daily against every approved listing's source URL and routes failures to a human admin queue within 24 hours. A listing whose source link returns an error is flagged "Needs recheck" — never silently hidden.

Reviews are user-submitted feedback, moderated before publishing, and shown separately from source-link verification. Reviews are opinions; verification is an audit. We do not let one substitute for the other.
