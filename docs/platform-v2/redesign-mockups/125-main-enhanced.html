Common questions

All FAQs →
Is USCEHub free?
Yes, browsing and filtering are free for everyone. A free account unlocks the weekly digest, saved searches, alerts, and synced shortlists across devices.
Who is USCEHub for?
Any medical student, resident, fellow, or attending looking for U.S. clinical experience — observerships, electives, externships, research. Listings cover visa-supported and US-only paths.
What does "Verified" mean?
A "Verified" listing has been admin-reviewed at publication and the institution's source URL has been confirmed live. Listings labeled "Official source on file" link to the institution's own page; broken links are flagged by users and triaged.
Which visa types support clinical experience?
B-1/B-2 (visitor) is the most common for observerships. F-1 OPT works for some externships. J-1 for funded research. The Visa helper tool walks through the decision flow.
Do you help me apply or write LORs?
No. We're a directory, not a placement service or LOR mill. Every listing routes you to the institution's own application path — no middleman, no fees.
Why do some listings have no cost or duration?
When the institution's page doesn't disclose a fee or fixed duration, we mark it "Contact" rather than guess. We never invent data.

How USCEHub verifies programs

Four-layer verification. We're conservative — listings are labeled "Verified" only after admin review, and we surface trust state on every card.

01 · Source

Official source link tracked

Every program links to the institution's own page — the URL the program controls. No aggregators, no scraped data.

02 · Review

Human admin review

Before publishing, a human reviewer confirms type, fees, visa notes, and that the source URL is the institution's own.

03 · Reports

Broken-link reporting

Any user can flag a dead link in one click. Reports route to admin triage; the source is re-checked manually.
