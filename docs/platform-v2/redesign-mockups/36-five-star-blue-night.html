How USCEHub works

A directory, not a placement service. We do not apply for you. We make sure the link works.

STEP 01 · SEARCH

Filter by what matters

Specialty, state, fee range, visa note, time of year. Results prioritize source-linked, recently reviewed listings.

STEP 02 · OPEN SOURCE

Use the institution's link

Every listing carries the institution's own application URL. Apply at the source. Save listings to revisit, compare up to three side by side.

STEP 03 · BUDGET

Estimate the trip

Use the cost estimator for housing, food, transport, and insurance. USMLE / ECFMG / visa / airfare fees are not included.

What "verified" actually means here

USCEHub is a structured directory. We confirm three specific things, in plain English. Anything we cannot confirm, we say so.

01 · SOURCE LINK

The URL works

An automated cron and human admins confirm the institution's application URL returns a working program page. Stale links are flagged "Needs recheck" — never silently hidden.

02 · ADMIN REVIEW

The listing is approved

An admin reviewed the listing copy for accuracy before publishing. We don't auto-publish poster submissions.

03 · REVIEWS SEPARATE

Opinions ≠ audits

Community reviews are user-submitted feedback, moderated before publishing, and shown separately from source-link verification. Reviews are opinions; verification is an audit.
