What "verified" means

Three things we confirm before a listing is published:

  1. The source link works. An automated cron and human admins confirm the institution's application URL returns a working program page.
  2. The listing is approved. An admin reviewed the listing copy for accuracy before publishing.
  3. Reviews are separate. Community reviews are user-submitted feedback shown separately from source-link verification.
