What "verified" means here

USCEHub is a directory, not a placement service. We confirm three specific things, in plain English. Anything we cannot confirm, we say so.

01 · SOURCE LINK

The URL works

An automated cron and human admins confirm the institution's application URL returns a working program page. Stale links are flagged "Needs recheck" — never silently hidden.

02 · ADMIN REVIEW

The listing is approved

An admin reviewed the listing copy for accuracy before publishing. We don't auto-publish poster submissions.

03 · REVIEWS SEPARATE

Opinions ≠ audits

Community reviews are user-submitted feedback shown separately from source-link verification. Reviews are opinions; verification is an audit.
