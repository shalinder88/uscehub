# UX Improvements Roadmap — Post-Launch

## Already Doing Well
- Ungated tools (no login required)
- Verified badges with source citations
- No stock photos
- Dark mode default
- Clean card-based layout

## Priority Improvements

### 1. Map Color Palette
- Current: red/orange/yellow/green (colorblind-unsafe)
- Better: single-hue blue sequential (5-7 stops)
- Use ColorBrewer 2.0 colorblind-safe palette
- Timeline: post-launch week 1

### 2. Dashboard Sidebar Navigation (Desktop)
- Current: horizontal scrolling tabs
- Better: collapsible left sidebar with grouped categories
- Mobile: bottom tab bar with 5 icons
- Timeline: post-launch week 2

### 3. Slide-in State Detail Panel
- Current: clicking state → navigates to new page
- Better: slide-in panel from right, keeping map visible
- Includes specialty filter above map
- Timeline: post-launch week 2

### 4. Inline Source Citations
- Current: verified badges at page level
- Better: source icon next to every individual data point
- Hover → "Source: NRMP 2025 SMS Report"
- Timeline: ongoing

### 5. Freshness Color Coding
- Current: "Last verified March 2026" (text only)
- Better: Green (<30 days), Yellow (30-90), Red (90+)
- Timeline: post-launch week 1

### 6. Client Page Meta Titles
- Several client components share generic "Career & Immigration" title
- Need unique titles for SEO
- Timeline: post-launch week 1

### 7. Employer CTA Optimization
- Add specific metrics: "X physicians searched for positions this month"
- Free tier: 1 listing free to reduce friction
- A/B test CTA wording
- Timeline: post-launch week 3
